# PDF.notes

Open PDFs directly in Obsidian and annotate them – drawings are saved directly into the PDF file.

## Features

- **Ink Drawing**: Draw directly on PDF pages with a pen or touch input.
- **Customizable Tools**:
  - Pen with adjustable size, color, and opacity.
  - Shapes (Rectangle, Ellipse, Line, Arrow).
  - Eraser (Stroke or Pixel mode).
- **Page Management**:
  - Insert new blank, lined, grid, dotted, or Cornell style pages.
  - Delete existing pages.
- **Auto-Save**: Everything you draw is automatically written into the PDF.
- **Flattening**: Permanently bake your drawings into the PDF so they can be viewed in ANY PDF viewer without the plugin.
- **Backups**: Optional automatic backups before any structural change (insert/delete).

## How to use

1. Open a PDF file in Obsidian.
2. Click the Brush icon in the ribbon or right-click the file and select **Open with PDF.notes**.
3. Use the toolbar at the top to select your tool and start drawing.
